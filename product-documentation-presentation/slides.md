# Algorithmic Complexity

We are optimizing our search algorithm to ensure sub-second response times.

The complexity of our new indexing algorithm is:

$$
T(n) = O(n \log n)
$$

Where:
- $n$ is the number of documentation pages.
- $\log n$ represents the tree traversal depth.
