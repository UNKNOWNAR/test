## Quick Start

### View Online
Visit the live presentation: **https://unknownar.github.io/test/**

### Run Locally

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd quarterly-earnings-presentation
   ```

2. **Start a local server**
   
   Using Python:
   ```bash
   python -m http.server 8000
   ```
   
   Or using Node.js:
   ```bash
   npm install
   npm start
   ```

3. **Open in browser**
   Navigate to `http://localhost:8000`

## Keyboard Shortcuts

- **F** - Fullscreen mode
- **S** - Speaker notes view
- **O** - Overview mode
- **Arrow Keys** - Navigate slides
- **ESC** - Exit overview/fullscreen
- **?** - Show help

## Presentation Structure

1. **Title Slide** - Introduction with contact information
2. **Executive Summary** - Markdown slide with key highlights
3. **Key Metrics** - Animated metric cards
4. **Risk Assessment** - Python code example
5. **Financial Formulas** - Mathematical equations
6. **Revenue Breakdown** - Animated bar chart
7. **Fraud Detection** - JavaScript code example
8. **Future Outlook** - Q1 2026 projections
9. **Q&A** - Closing slide

## Technologies Used

- [RevealJS 4.5.0](https://revealjs.com/) - Presentation framework
- [KaTeX](https://katex.org/) - Mathematical equation rendering
- [Highlight.js](https://highlightjs.org/) - Code syntax highlighting
- Custom CSS with modern design patterns

## Speaker Notes

Press **S** during the presentation to open the speaker notes view. Each slide includes detailed notes to guide the presenter through key talking points.
